Tipos Condiciones

Var - General - No recomendable
(Igual que Let, excepto se puede declarar más veces donde sea)


var edad = 10;
var edad = 20;

console.log( edad );

Let - Local
(Igual que Const, excepto se puede declarar más veces en diferentes condiciones y cambiar valor)


let edad = 10;
edad = 20;

console.log( edad );

Const - Local
(Igual que Let, excepto no se puede declarar más veces, ni cambiar valor en la misma condición, excepto en array y objetos)


const edad = 10;

console.log( edad );
