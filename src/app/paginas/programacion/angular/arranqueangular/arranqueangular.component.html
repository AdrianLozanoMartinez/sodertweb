Angular - Arranque
  • Navegador
    Verlo después de arrancar
  • Simular servidor
  • Preparar para servidor
  • Defecto: localhost:4200
  • Abrir más: localhost:puerto_que_elijamos
  • ng serve: Solo compila, se muestra si ya tenemos cargado la página en el navegador (si acaso dar a actualizada)
  • ng serve -o (-open): Compila + Abre navegador
  • npm start: Compila + Abre navegador
  • ng serve --port 4200 (--port puerto_que_elijamos): Compila en otro puerto, para cuando haya conflictos
  • ng serve --port 4200 -o (--port puerto_que_elijamos -open): Compila en otro puerto + Abre navegador
  • ng build: Compila para DESARROLLO (Enviroment.ts)
  • ng build --prod=true: Compila para PRODUCCIÓN (Enviroment.prod.ts)
  • Carpeta Dist: COPIAR su contenido en la Carpeta_proyecto del Servidor -> C:/wamp64/www/carpeta_proyecto
