Instalación Automático

  1. Instalar: npm install bootstrap --save
    +
    Añadir en angular.json:
    "styles": [
"./node_modules/bootstrap/dist/css/bootstrap.min.css",
"src/styles.css"
]
  2. Instalar: npm install jquery --save
          npm install popper.js --save
    +
    Añadir en angular.json:
    "scripts": [
    "./node_modules/jquery/dist/jquery.min.js",
    "./node_modules/popper.js/dist/umd/popper.min.js",
    "./node_modules/bootstrap/dist/js/bootstrap.min.js"
]

Descargada

Lo que descargamos en la web lo metemos en:

src/assets/Bootstrap-4.5.0-Descargado/css
y en
src/assets/Bootstrap-4.5.0-Descargado/js
