  • Estructura interna:
    • Css: Estilo
    • Html: Visual
    • Ts (Typescript):
      • Import: Para usar servicios, interface, otros dentro del componente
      • Decorador (@Component): Usar decoradores (stylos y otros) en todo el componente
      • Export: Para poder importar en otro componente... Dentro de él declaramos variables, constructor y funciones
      • Constructor: Dentro de él declaramos variables, funciones... 1º en ejecutarse
      • ngOnInit: Dentro de él declaramos variables, funciones... 2º en ejecutarse
    • Spec.ts: Test de typescript

  //Componentes - Páginas añadir después de crearlas
  {{c}} path: '', component: HomeComponent },                        //Siempre va a HomeComponent cuando da errores los botones

  //Componentes - Especiales
  {{c}} path: 'appcomponente', component: AppComponent },
  //Componentes - Páginas añadir después de crearlas - Componentes Error - Opción 1

  //Componentes - Páginas añadir después de crearlas - Componentes Error - Opción 2
  {{c}} path: '**', pathMatch: 'full', redirectTo: 'home'},           //Si introducimos mal en la barra de direcciones nos va al HomeComponent (home)
