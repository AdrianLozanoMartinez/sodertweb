Var

  • Sustituye el valor de la variable con el mismo nombre, aunque esté dentro de un if o función

Let

  • Diferente valor de la variable con el mismo nombre, aunque esté dentro y fuera del if o función

Const

  • Valor único, no se cambia y se usa cuando deseamos su valor permanente, pesa poco y carga rápido
